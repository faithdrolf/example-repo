# Simple Python Calculator
## A simple Git version‑control practice project.
### Description
This project is a menu‑driven Python calculator that allows users to perform basic arithmetic operations and view previously completed equations. The program supports addition, subtraction, multiplication, and division, while also handling invalid input and division‑by‑zero errors gracefully.

Each completed calculation is saved to a file named equations.txt, and users can choose to view their full history of equations at any time through the menu system.
